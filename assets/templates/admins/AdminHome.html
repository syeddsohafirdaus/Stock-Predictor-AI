System Architecture

This platform uses an attention-based LSTM model to predict International Stock Price trends. The system consists of five core layers:

  • 🔐 Shell Layer: User authentication & security
  • 📥 Input Layer: Data validation & preprocessing
  • 🧠 Hidden Layer: LSTM neural network processing
  • 👁️ Attention Layer: Trend prediction generation
  • 📤 Output Layer: Results presentation
  • ⚡ ML Stack: TensorFlow, XGBoost, Scikit-learn
