Dataset Schema

The dataset contains historical indicators necessary for technical training:

  • 📅 Date: Session trading date
  • 📈 Open: Initial transaction price
  • 🚀 High: Maximum session price
  • 📉 Low: Minimum session price
  • 🏁 Close: Final market closing price
  • 📦 Volume: Number of shares traded
💡 Training Setup

This CSV is automatically partitioned into training (65%) and testing (35%) divisions. It generates moving averages (7/21 days), MACD, Bollinger Bands, and Fourier components before initiating model training.
